# Khushbu Parmar — Odoo Portfolio

A responsive digital portfolio for Odoo, Python and backend development work.

## Preview locally

Open `index.html` directly in a browser, or use VS Code Live Server.

## Publish with GitHub Pages

1. Create a new public repository named `odoo-portfolio`.
2. Upload all files from this folder, including `assets`.
3. Open repository **Settings → Pages**.
4. Under **Build and deployment**, select **Deploy from a branch**.
5. Select branch `main` and folder `/ (root)`.
6. Save and wait 1–3 minutes.
7. Open: `https://khushbuparmar2029.github.io/odoo-portfolio/`

## Important

The `index.html` file must be in the repository root, not inside another nested folder.
